# physical

## To build a development enviornment:

1. install pip(pip3) first.

2. execute

```shell
$ pip3 install django
$ git clone https://github.com/Marzear/physical.git
$ cd physical
$ python manage.py migrate
```

## To run the server

```
$ python manage.py runserver
```
